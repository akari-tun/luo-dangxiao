## 一、项目概述

### 1.1 项目信息
- **项目名称**: 湖南省委党校自助服务系统
- **子系统**: 卡务中心 (CardCenter)
- **技术平台**: Avalonia UI + CommunityToolkit.Mvvm + .NET 10.0
- **应用类型**: 管理终端
- **用户类型**: 管理员、卡务工作人员
- **使用场景**: 卡片管理、充值退款、退卡、读卡等管理操作

### 1.2 整体设计规范

| 设计元素 | 规范 |
|----------|------|
| **主色调** | 红色系 (#C42828, #d9230b) |
| **辅助色** | 金黄色 (#e6b84d, #ffe3a3) |
| **内容区背景** | 米黄色/奶油色渐变 (#faf9f3) |
| **字体** | 中文字体：黑体 (SimHei) |
| **英文字体** | Arial |

## 二、系统架构

### 2.1 功能架构

```
卡务中心
├── 登录认证
│   └── 账号密码登录
│
├── 卡片管理
│   ├── 重置有效期
│   ├── 重置卡类
│   ├── 充值退款
│   ├── 退卡
│   └── 读卡
│
└── 系统管理
    ├── 操作日志
    └── 数据查询
```

### 2.2 页面流程

```
登录页面 → 主界面 → 功能选择 → 具体操作 → 完成
              ↓
         卡片管理界面
              ↓
    ┌─────────┼─────────┐
    ↓         ↓         ↓
 重置有效期  充值退款   退卡
 重置卡类    读卡
```

## 三、页面设计

### 3.1 登录页面 (CardCenterLogin)

**功能**: 管理员登录卡务中心

**元素**:
- 用户名输入框
- 密码输入框
- 登录按钮
- 记住密码选项

### 3.2 卡务中心主界面 (CardCenterHome)

**功能**: 显示所有卡片管理功能入口

**布局**: 网格功能菜单

**功能按钮**:
- 重置有效期
- 重置卡类
- 充值退款
- 退卡
- 读卡
- 操作日志

### 3.3 重置有效期界面 (ResetValidityPage)

**功能**: 修改卡片有效期

**流程**:
1. 读卡
2. 显示当前有效期
3. 输入新有效期
4. 确认重置

### 3.4 重置卡类界面 (ResetCardTypePage)

**功能**: 修改卡片类型

**流程**:
1. 读卡
2. 显示当前卡类
3. 选择新卡类
4. 确认重置

### 3.5 充值退款界面 (RefundPage)

**功能**: 卡片充值和退款

**流程**:
1. 读卡
2. 显示当前余额
3. 选择操作类型（充值/退款）
4. 输入金额
5. 确认操作

### 3.6 退卡界面 (ReturnCardPage)

**功能**: 退卡处理

**流程**:
1. 读卡
2. 显示卡片信息
3. 显示余额
4. 确认退卡
5. 处理退款

### 3.7 读卡界面 (ReadCardPage)

**功能**: 读取并显示卡片完整信息

**显示信息**:
- 卡号
- 持卡人信息
- 卡片状态
- 有效期
- 余额
- 交易记录
