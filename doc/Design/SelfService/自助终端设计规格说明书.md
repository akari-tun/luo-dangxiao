## 二、页面架构与关系

### 2.1 页面分类

系统页面分为三类：

#### (1) 主页面

| 页面 | 代码 | 说明 |
|------|------|------|
| **HomePage** | HomePage | 主界面，作为功能子页面的容器 |

#### (2) 功能子页面

| 页面 | 代码 | 说明 |
|------|------|------|
| **验证页面** | VerifyPage | 统一的身份验证入口 |
| **取卡页面** | TakeCardPage | 自助取卡功能 |
| **挂失页面** | ReportLossPage | 自助挂失功能 |
| **补卡页面** | ReplacementPage | 自助补卡功能 |
| **查询页面** | QueryPage | 自助查询功能 |
| **充值页面** | RechargePage | 自助充值功能 |
| **二维码页面** | QRCodePage | 二维码展示（支付等） |
| **报到页面** | CheckInPage | 自助报到功能 |
| **卡片处理页面** | CardProcessPage | 卡片操作过程展示 |
| **完成页面** | CompletionPage | 操作完成展示 |

#### (3) 模块页面

| 页面 | 代码 | 说明 |
|------|------|------|
| **学员信息模块** | StudentInfoPage | 学员信息展示模块 |
| **教职工信息模块** | StaffInfoPage | 教职工信息展示模块 |
| **身份证验证模块** | IDCardVerifyPage | 身份证验证模块 |
| **短信验证模块** | SMSVerifyPage | 短信验证模块 |

### 2.2 主页面状态

**HomePage** 具有三种状态：

1. **教职工自助状态** (`StaffSelfService`)
   - 显示教职工功能按钮（取卡、挂失、补卡、查询、充值）
   - 根据配置决定是否显示

2. **学员自助状态** (`StudentSelfService`)
   - 显示学员功能按钮（报到、取卡、挂失、补卡）
   - 根据配置决定是否显示

3. **子页面容器状态** (`SubPageContainer`)
   - 隐藏功能按钮菜单
   - 显示子页面容器区域
   - 加载对应的功能子页面

**状态转换**:
```
教职工自助状态/学员自助状态 
    ↓ 用户点击功能按钮
子页面容器状态（加载对应功能子页面）
    ↓ 操作完成或返回
教职工自助状态/学员自助状态
```

### 2.3 页面加载机制

1. **主页面作为容器**
   - HomePage 作为整个自助服务的主容器
   - 包含顶部栏（Logo、标题、倒计时）
   - 包含底部按钮区域
   - 中间区域为子页面容器

2. **功能子页面加载**
   - 功能子页面（VerifyPage、TakeCardPage等）加载到 HomePage 的子页面容器中
   - 子页面与 HomePage 共享顶部栏和底部按钮区域
   - 子页面根据用户类型加载对应的信息模块

3. **模块页面嵌入**
   - StudentInfoPage / StaffInfoPage 作为模块嵌入到需要展示用户信息的子页面中
   - IDCardVerifyPage / SMSVerifyPage 作为验证模块嵌入到 VerifyPage 中

## 三、功能流程

### 3.1 教职工自助流程

```
┌─────────────────────────────────────────────────────────────┐
│                      教职工自助流程                          │
└─────────────────────────────────────────────────────────────┘

【自助取卡】
HomePage(教职工自助) → VerifyPage → TakeCardPage → CardProcessPage → CompletionPage
                           ↓              ↓              ↓
                    (嵌入验证模块)   (嵌入StaffInfo   (制卡过程)
                     IDCardVerifyPage   Page展示教工
                     或SMSVerifyPage    信息)

【自助挂失】
HomePage(教职工自助) → VerifyPage → ReportLossPage → CardProcessPage → CompletionPage
                           ↓              ↓
                    (嵌入验证模块)   (嵌入StaffInfo
                     IDCardVerifyPage   Page展示教工
                     或SMSVerifyPage    信息)

【自助补卡】
HomePage(教职工自助) → VerifyPage → ReplacementPage → CardProcessPage → CompletionPage
                           ↓              ↓
                    (嵌入验证模块)   (嵌入StaffInfo
                     IDCardVerifyPage   Page展示教工
                     或SMSVerifyPage    信息)

【自助查询】
HomePage(教职工自助) → VerifyPage → QueryPage
                           ↓         ↓
                    (嵌入验证模块)  (嵌入StaffInfo
                     IDCardVerifyPage  Page展示教工
                     或SMSVerifyPage   信息及查询结果)

【自助充值】
HomePage(教职工自助) → VerifyPage → RechargePage → QRCodePage → CardProcessPage → CompletionPage
                           ↓           ↓            ↓
                    (嵌入验证模块)  (金额选择)   (支付二维码)
                     IDCardVerifyPage  (嵌入StaffInfo  (嵌入StaffInfo
                     或SMSVerifyPage    Page)           Page)
```

### 3.2 学员自助流程

```
┌─────────────────────────────────────────────────────────────┐
│                      学员自助流程                            │
└─────────────────────────────────────────────────────────────┘

【自助取卡】
HomePage(学员自助) → VerifyPage → TakeCardPage → CardProcessPage → CompletionPage
                        ↓              ↓              ↓
                 (嵌入验证模块)   (嵌入StudentInfo (制卡过程)
                  IDCardVerifyPage   Page展示学员
                  或SMSVerifyPage    信息)

【自助挂失】
HomePage(学员自助) → VerifyPage → ReportLossPage → CardProcessPage → CompletionPage
                        ↓              ↓
                 (嵌入验证模块)   (嵌入StudentInfo
                  IDCardVerifyPage   Page展示学员
                  或SMSVerifyPage    信息)

【自助补卡】
HomePage(学员自助) → VerifyPage → ReplacementPage → CardProcessPage → CompletionPage
                        ↓              ↓
                 (嵌入验证模块)   (嵌入StudentInfo
                  IDCardVerifyPage   Page展示学员
                  或SMSVerifyPage    信息)

【自助报到】
HomePage(学员自助) → VerifyPage → CheckInPage → CompletionPage
                        ↓           ↓
                 (嵌入验证模块)  (嵌入StudentInfo
                  IDCardVerifyPage   Page展示学员
                  或SMSVerifyPage    信息及报到状态)
```

### 3.3 页面流程说明

1. **所有功能均以 VerifyPage 开始**
   - 统一的身份验证入口
   - 根据系统参数决定默认验证方式（身份证/短信）
   - 支持在验证页面内切换验证方式

2. **验证通过后进入功能页面**
   - 根据选择的功能加载对应的功能子页面
   - 功能子页面内嵌对应的信息展示模块（StudentInfoPage/StaffInfoPage）

3. **需要卡片操作的功能包含 CardProcessPage**
   - 取卡、挂失、补卡等操作需要展示卡片处理过程
   - CardProcessPage 展示实时操作状态

4. **充值功能包含 QRCodePage**
   - 展示支付二维码
   - 用户扫码支付后进入 CardProcessPage 完成充值

5. **所有流程以 CompletionPage 结束**
   - 展示操作结果
   - 显示相关信息
   - 提供退出按钮返回主界面

## 四、页面设计详解

### 4.1 主界面 (HomePage)

**三种状态**：
1. **教职工自助状态** - 显示教职工功能按钮
2. **学员自助状态** - 显示学员功能按钮
3. **子页面容器状态** - 隐藏功能按钮，显示子页面容器

**状态切换**：
- 初始状态根据配置显示教职工或学员自助界面
- 点击功能按钮后切换到子页面容器状态
- 操作完成或返回后恢复到初始状态

### 4.2 验证页面 (VerifyPage)

**功能**：统一的身份验证入口

**嵌入模块**：
- IDCardVerifyPage（身份证验证模块）
- SMSVerifyPage（短信验证模块）

**交互**：
- 支持验证方式切换
- 验证成功后进入对应功能子页面

### 4.3 功能子页面

**通用特性**：
- 加载到 HomePage 的子页面容器中
- 共享 HomePage 的顶部栏（含倒计时）和底部按钮
- 根据用户类型嵌入 StudentInfoPage 或 StaffInfoPage

**列表**：
- **TakeCardPage** - 取卡操作
- **ReportLossPage** - 挂失操作
- **ReplacementPage** - 补卡操作
- **QueryPage** - 查询操作
- **RechargePage** - 充值操作
- **QRCodePage** - 二维码展示
- **CheckInPage** - 报到操作
- **CardProcessPage** - 卡片处理过程
- **CompletionPage** - 完成展示

### 4.4 模块页面

**信息展示模块**：
- **StudentInfoPage** - 学员信息（姓名、班级、房间、缴费信息等）
- **StaffInfoPage** - 教职工信息（姓名、部门、卡号、余额等）

**验证模块**：
- **IDCardVerifyPage** - 身份证读取和验证
- **SMSVerifyPage** - 短信验证码发送和验证
