## 四、功能描述

### 4.1 页面用途

自助挂失界面用于用户自助挂失丢失的卡片。用户确认挂失后，系统将立即挂失卡片，卡片将失效无法使用。

### 4.2 核心功能

1. **卡片信息展示**
   - 显示用户卡片信息
   - 显示当前卡片状态
   - 显示卡片余额

2. **挂失确认**
   - 明确提示挂失后果
   - 二次确认机制
   - 防止误操作

3. **挂失执行**
   - 调用挂失接口
   - 更新卡片状态
   - 记录挂失日志

4. **完成确认**
   - 显示挂失成功提示
   - 提示补办新卡选项
   - 进入完成界面

### 4.3 操作流程

```
显示卡片信息 → 确认挂失 → 处理中 → 挂失成功 → 完成
      ↓             ↓          ↓
  取消返回    已挂失状态   挂失失败
      ↓             ↓          ↓
  返回上一页  提示已挂失   提示重试
```

## 五、交互描述

### 5.1 状态转换

```
状态 1: 确认挂失
    ↓ 点击"确认挂失"
状态 2: 处理中
    ↓ 处理完成
状态 3: 挂失成功
    ↓ 点击"完成"
完成界面
```

### 5.2 详细交互

**1. 页面加载 (状态 1)**:
- 从后台获取用户卡片信息
- 验证卡片状态
- 如已挂失，提示"卡片已挂失"
- 启动倒计时计时器

**2. 确认挂失**:
- 用户点击"确认挂失"
- 验证卡片状态为"正常"
- 显示警告提示
- 进入状态 2 (处理中)

**3. 挂失处理 (状态 2)**:
- 发送挂失请求到后台
- 显示处理动画
- 等待处理结果
- 禁用返回按钮

**4. 挂失完成 (状态 3)**:
- 返回挂失成功
- 显示挂失成功提示
- 显示挂失信息和时间
- 提示可补办新卡

**5. 完成**:
- 用户点击"完成"
- 进入完成界面
- 可选择补卡功能

**6. 异常处理**:
- **已挂失**: 提示"该卡片已挂失，无需重复挂失"
- **挂失失败**: 提示"挂失失败，请重试"
- **网络错误**: 提示"网络连接失败，请重试"

## 七、实现注意事项

### 7.1 ViewModel 实现

```csharp
public partial class ReportLossViewModel : ViewModelBase
{
    [ObservableProperty]
    private CardInfo _cardInfo;
    
    [ObservableProperty]
    private ReportLossState _currentState;
    
    [ObservableProperty]
    private string _statusMessage;
    
    [ObservableProperty]
    private bool _canGoBack;
    
    public void Initialize(CardInfo card)
    {
        CardInfo = card;
        CurrentState = ReportLossState.Confirm;
        
        // 检查卡片状态
        if (card.Status == CardStatus.Lost)
        {
            StatusMessage = "该卡片已挂失";
            // 禁用确认按钮
        }
    }
    
    [RelayCommand]
    private async Task ConfirmReportLossAsync()
    {
        if (CardInfo.Status == CardStatus.Lost)
        {
            await ShowMessageAsync("该卡片已挂失，无需重复挂失");
            return;
        }
        
        CurrentState = ReportLossState.Processing;
        CanGoBack = false;
        
        try
        {
            var result = await CardService.ReportLossAsync(CardInfo.CardNumber);
            
            if (result.Success)
            {
                CardInfo.Status = CardStatus.Lost;
                CardInfo.LossTime = DateTime.Now;
                CurrentState = ReportLossState.Completed;
            }
            else
            {
                await ShowErrorAsync(result.Message);
                CurrentState = ReportLossState.Confirm;
                CanGoBack = true;
            }
        }
        catch (Exception ex)
        {
            await ShowErrorAsync("挂失失败，请重试");
            CurrentState = ReportLossState.Confirm;
            CanGoBack = true;
        }
    }
}

public enum ReportLossState
{
    Confirm,     // 确认挂失
    Processing,  // 处理中
    Completed    // 完成
}
```
