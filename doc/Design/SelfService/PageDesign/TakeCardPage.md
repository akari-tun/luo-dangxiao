## 四、功能描述

### 4.1 页面用途

自助取卡界面用于指导用户完成卡片领取操作。界面展示用户信息，确认取卡意愿，展示制卡过程，提示用户领取卡片，并最终确认取卡完成。

### 4.2 核心功能

1. **信息确认**
   - 展示用户基本信息
   - 显示卡片信息
   - 确认取卡操作

2. **制卡控制**
   - 发送制卡指令到设备
   - 显示制卡进度
   - 处理制卡异常

3. **出卡检测**
   - 检测卡片是否制作完成
   - 检测用户是否已取走卡片
   - 超时未取卡处理

4. **完成确认**
   - 确认取卡完成
   - 更新系统状态
   - 进入完成界面

### 4.3 操作流程

```
信息确认 → 点击确认取卡 → 制卡中 → 出卡提示 → 取卡检测 → 完成 → 进入完成界面
    ↓              ↓              ↓           ↓            ↓
 取消返回      制卡失败      设备故障    超时未取      取卡异常
    ↓              ↓              ↓           ↓            ↓
 返回上一页    提示重试      联系管理员  回收卡片      异常处理
```

## 五、交互描述

### 5.1 状态转换

```
状态 1: 确认取卡
    ↓ 点击"确认取卡"
状态 2: 制卡中
    ↓ 制卡完成
状态 3: 出卡提示
    ↓ 检测到取卡
状态 4: 取卡完成
    ↓ 点击"完成"
完成界面
```

### 5.2 详细交互

**1. 页面加载 (状态 1)**:
- 从后台获取用户卡片信息
- 显示用户信息和卡片状态
- 启动倒计时计时器

**2. 确认取卡**:
- 用户点击"确认取卡"
- 验证卡片状态为"待领取"
- 进入状态 2 (制卡中)

**3. 制卡过程 (状态 2)**:
- 发送制卡指令到制卡设备
- 显示制卡动画和提示
- 等待制卡完成
- 禁用返回按钮

**4. 制卡完成**:
- 设备返回制卡成功
- 进入状态 3 (出卡提示)
- 卡片已到达出卡口

**5. 出卡提示 (状态 3)**:
- 显示取卡动画和箭头
- 提示用户从出卡口取卡
- 检测传感器状态

**6. 取卡检测**:
- 检测用户是否已取走卡片
- 成功：进入状态 4 (完成)
- 超时：提示超时，回收卡片

**7. 完成 (状态 4)**:
- 显示取卡成功提示
- 显示取卡信息和时间
- 用户点击"完成"
- 进入完成界面

**8. 异常处理**:
- **制卡失败**: 提示"制卡失败，请重试"，返回状态 1
- **设备故障**: 提示"设备故障，请联系管理员"
- **超时未取**: 回收卡片，提示"取卡超时，请重新操作"

### 5.3 硬件交互

public interface ICardPrinter
{
    Task<PrintResult> PrintCardAsync(CardData data);

    Task<bool> IsCardTakenAsync();        // 检测用户是否已取走卡片
    Task<bool> EjectCardAsync();          // 弹出卡片
    Task<bool> RetractCardAsync();        // 回收卡片
}

public class CardData
{
    public string CardNumber { get; set; }
    public string Name { get; set; }
    public string ClassName { get; set; }
    public DateTime ValidFrom { get; set; }
    public DateTime ValidTo { get; set; }
}

## 七、实现注意事项

### 7.1 硬件状态管理

```csharp
public enum CardMachineState
{
    Idle,           // 空闲
    Printing,       // 制卡中
    CardReady,      // 卡片就绪
    CardTaken,      // 已取卡
    CardRetracted,  // 已回收
    Error           // 故障
}

public partial class TakeCardViewModel : ViewModelBase
{
    [ObservableProperty]
    private CardMachineState _machineState;
    
    [ObservableProperty]
    private string _statusMessage;
    
    [ObservableProperty]
    private bool _canGoBack;
    
    partial void OnMachineStateChanged(CardMachineState value)
    {
        switch (value)
        {
            case CardMachineState.Idle:
                StatusMessage = "请确认取卡";
                CanGoBack = true;
                break;
            case CardMachineState.Printing:
                StatusMessage = "正在制卡，请稍候...";
                CanGoBack = false;
                break;
            case CardMachineState.CardReady:
                StatusMessage = "请取走您的卡片";
                CanGoBack = false;
                break;
            case CardMachineState.CardTaken:
                StatusMessage = "取卡完成";
                CanGoBack = false;
                break;
            case CardMachineState.Error:
                StatusMessage = "设备故障";
                CanGoBack = true;
                break;
        }
    }
}
```

### 7.2 传感器检测

```csharp
// 检测卡片是否被取走
private async Task WaitForCardTakenAsync()
{
    var timeout = TimeSpan.FromSeconds(30);  // 30秒超时
    var startTime = DateTime.Now;
    
    while (DateTime.Now - startTime < timeout)
    {
        if (await CardPrinter.IsCardTakenAsync())
        {
            MachineState = CardMachineState.CardTaken;
            return;
        }
        
        await Task.Delay(500);  // 每500ms检测一次
    }
    
    // 超时，回收卡片
    await CardPrinter.RetractCardAsync();
    await ShowTimeoutMessageAsync();
}
```
