## 四、功能描述

### 4.1 页面用途

报到页面用于学员完成自助报到操作。页面展示学员信息、培训信息、报到状态，学员确认信息后完成报到，系统分配房间并更新报到状态。

### 4.2 核心功能

1. **信息展示**
   - 嵌入 StudentInfoPage 展示学员信息
   - 显示培训班信息
   - 显示当前报到状态

2. **报到处理**
   - 提交报到请求到后台
   - 系统分配房间号
   - 更新报到状态和时间

3. **状态管理**
   - 检测学员是否已报到
   - 已报到学员显示提示信息
   - 防止重复报到

4. **结果展示**
   - 报到成功后显示房间号
   - 报到时间记录
   - 提供入住指引

### 4.3 报到流程

```
验证通过 → 显示学员信息 → 确认报到 → 处理中
                                              ↓
                                        分配房间
                                              ↓
                                        更新状态
                                              ↓
                                        报到完成 → 显示房间号 → 完成
```

## 五、交互描述

### 5.1 状态转换

```
状态 1: 报到确认
    ↓ 点击"确认报到"（未报到）
状态 2: 报到处理中
    ↓ 处理完成
状态 3: 报到完成
    ↓ 点击"完成"
完成界面 (CompletionPage)

或

状态 1: 报到确认
    ↓ 检测到已报到
状态 4: 已报到提示
    ↓ 点击"返回主页"
HomePage
```

### 5.2 详细交互

**1. 页面加载**:
- 获取学员信息
- 嵌入 StudentInfoPage 模块
- 查询报到状态
- 如果已报到，显示状态 4
- 如果未报到，显示状态 1
- 启动倒计时计时器

**2. 报到确认**:
- 展示学员信息和培训信息
- 显示当前报到状态（未报到）
- 显示房间号（待分配）
- 用户确认信息无误后点击"确认报到"

**3. 报到处理（状态 2）**:
- 显示处理动画
- 禁用返回按钮
- 提交报到请求到后台
- 后台处理：分配房间、更新状态
- 等待处理完成

**4. 报到完成（状态 3）**:
- 显示报到成功图标
- 更新报到状态为"已报到"
- 显示报到时间
- 显示分配的房间号（高亮显示）
- 提示用户前往房间入住
- 用户点击"完成"进入完成界面

**5. 已报到处理（状态 4）**:
- 页面加载时检测到学员已报到
- 显示提示"您已完成报到"
- 显示之前的报到信息和房间号
- 提示"无需重复操作"
- 提供"返回主页"按钮

**6. 异常处理**:
- **报到失败**: 提示"报到失败，请重试"，返回状态 1
- **网络异常**: 提示"网络异常，请检查网络连接后重试"
- **房间分配失败**: 提示"房间分配失败，请联系管理员"

### 5.3 数据模型

## 七、实现注意事项

### 7.1 ViewModel 实现

```csharp
public partial class CheckInViewModel : ViewModelBase
{
    [ObservableProperty]
    private StudentInfo _studentInfo;
    
    [ObservableProperty]
    private CheckInInfo _checkInInfo;
    
    [ObservableProperty]
    private CheckInPageState _currentState;
    
    [ObservableProperty]
    private bool _canGoBack;
    
    public enum CheckInPageState
    {
        Confirm,      // 报到确认
        Processing,   // 处理中
        Completed,    // 完成
        AlreadyDone   // 已报到
    }
    
    partial void OnCurrentStateChanged(CheckInPageState value)
    {
        CanGoBack = value == CheckInPageState.Confirm || value == CheckInPageState.AlreadyDone;
    }
    
    [RelayCommand]
    private async Task ConfirmCheckInAsync()
    {
        CurrentState = CheckInPageState.Processing;
        
        var result = await CheckInService.CheckInAsync(StudentInfo.Id);
        
        if (result.Success)
        {
            CheckInInfo.Status = CheckInStatus.CheckedIn;
            CheckInInfo.RoomNumber = result.RoomNumber;
            CheckInInfo.CheckInTime = result.CheckInTime;
            CurrentState = CheckInPageState.Completed;
        }
        else
        {
            await ShowErrorAsync(result.ErrorMessage);
            CurrentState = CheckInPageState.Confirm;
        }
    }
}
```

### 7.2 报到状态检测

```csharp
private async Task LoadCheckInStatusAsync()
{
    var checkInInfo = await CheckInService.GetCheckInInfoAsync(StudentInfo.Id);
    
    if (checkInInfo.Status == CheckInStatus.CheckedIn)
    {
        // 已报到，显示状态 4
        CurrentState = CheckInPageState.AlreadyDone;
    }
    else
    {
        // 未报到，显示状态 1
        CurrentState = CheckInPageState.Confirm;
    }
}
```
